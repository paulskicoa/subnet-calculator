Enter an IP address in CIDR notation (for example, 192.42.53.12/24) below to get the associated Network ID and address range.

If the CIDR value isn't known, use the classful addressing scheme. Look at the first number in the IP address and find the corresponding CIDR value below.
0 to 127 is Class A. Use /8.
128 to 191 is Class B. Use /16.
192 to 223 is Class C. Use /24.

How many networks (subnets) do you need in your address space? If you aren't sure, you can leave this at the default of 1.

The number you choose will be rounded up to the nearest power of 2 to ensure there's an equal number of maximum hosts on all subnets. For example, if you select 6, you'll get 8 subnets and could choose any 6 of these to use. The remaining subnets would be available if needed at a later date. Remember that the first and last address in each subnet are reserved for the Network ID and Broadcast Address, respectively.
